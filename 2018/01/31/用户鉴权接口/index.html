业务对象

UserObject：用户

名称 类型 说明
id String 用户ID
username String 用户账号，统一使用邮箱地址
nickname String 昵称
sex Integer 性别，0-女，1-男
age Integer 年龄
avatarUrl String 头像URL
area !AreaObject 注册时所在的位置信息，参考PM25项目中对该类型的定义
regfrom String 注册来源，pm25、wifi等，暂定默认为pm25，来自智能纽扣为：pedobutton

DeviceObject：用户设备

名称 类型 说明 可为空
id Integer 设备id 否
type Integer 设备类型 否

AreaObject：地区对象

名称 类型 说明
id String 地区id
name String 名称
pinyin String 包含JsonArray的字串。名称小写全拼数组。例如：“朝阳区”为[ “chao”,”yang”,”qu” ]
parentId String 上一层地区id
rootId String 所属省或直辖市id
level String 级别名字：sheng（省级），di（地级），xian（县级）
poi !PoiObject 地图上的点

PoiObject：地图上的点

名称 类型 说明
latitude Float 纬度
longitude Float 经度
description String 该点的逆地理描述信息

接口

验证码发送接口

  • url：http://api.aigolife.com/sendVerifyCode.jhtml
  • 必须参数：sign:md5(接口完整url包含传入的参数(不包含参数sign)+’2013-03-29’+action).toLowerCase()

1.发送短信验证码：sendVerifyCode

  • 传入：
名称 类型 说明 可为空
action String 传入值固定为(‘sendVerifyCode’) 否
phone String 手机号码 否
sign String 必须拼在接口参数的最后面 生成规则：md5(“http://api.life.aigo.com/sendVerifyCode.html?action=sendVerifyCode&phone=15850754914"+"2013-03-29"+action).toLowerCase() 其中action 值固定为字符串’sendVerifyCode’ 否
  • 返回：
名称 类型 说明 可为空
result ResultObject 执行结果 否
identifier String 请求验证码成功时返回 否
  • 接口地址：http://api.aigolife.com/sendVerifyCode.jhtml
  • 实例：http://api.life.aigo.com/sendVerifyCode.html?action=sendVerifyCode&phone=15850754914&sign=5ec63a74d7dee56c21e4e9ede6f15703

2.发送邮箱验证码：sendVerifyCode

  • 传入：
名称 类型 说明 可为空
action String 传入值固定为(‘sendVerifyCode’) 否
mail String 邮箱地址 否
sign String 必须拼在接口参数的最后面 生成规则：md5(“http://api.life.aigo.com/sendVerifyCode.html?action=sendVerifyCode&mail=username@gmail.com"+"2013-03-29"+action).toLowerCase() 其中action 值固定为字符串’sendVerifyCode’ 否
  • 返回：
名称 类型 说明 可为空
result ResultObject 执行结果 否
identifier String 请求验证码成功时返回 否
  • 接口地址：http://api.aigolife.com/sendVerifyCode.jhtml
  • 实例：http://api.life.aigo.com/sendVerifyCode.html?action=sendVerifyCode&mail=username@gmail.com&sign=根据规则计算获得

用户接口

  • url：http://api.aigolife.com/if/account.jhtml
  • 必须参数：sign:md5(接口完整url包含传入的参数(不包含参数sign)+’2013-03-29’+action).toLowerCase()
  • sign要求：sign参数必须拼接在接口的最后
  • sign参数实例：假设调用接口地址是http://api.aigolife.com/if/account.jhtml?action=UserLogin&username=username123@163.com&password=e10adc3949ba5yabbe58e057j20f883d&sign=95e7aaeab675c5e9557s973edf545d43,那么sign生成规则为：md5Encode("http://api.aigolife.com/if/account.jhtml?action=UserLogin&username=username123@163.com&password=e10adc3949ba5yabbe58e057j20f883d"+"2013-03-29"+"UserLogin").toLowerCase()

1.登录：UserLogin

  • 传入：
名称 类型 说明 可为空
action String 传入值固定为(‘UserLogin’) 否
username String 用户名 否
password(md5(登录密码)) String 对用户登录密码进行md5加密 否
deviceId String 设备唯一标识,以应用于多终端登录同一个账户 是
sign String 必须拼在接口参数的最后面 生成规则：md5(“http://api.aigolife.com/if/account.jhtml?action=UserLogin&username=(登录的用户名)&password=(进行MD5加密后的登录密码)"+"2013-03-29"+action).toLowerCase() 其中action 值固定为字符串’UserLogin’ 否
  • 返回：
名称 类型 说明 可为空
result ResultObject 执行结果 否
token String 登录成功后返回的字符串，用于用户进行其他操作，也用于验证用户登录的有效性 否
expire DateTime：10位时间戳 登录成功的时间 否
duration Integer：秒 连续在线的有效时间 否
period Integer：秒 连续在线的有效时间（同上，可以忽略） 否
  • 接口地址：http://api.aigolife.com/if/account.jhtml
  • 实例：http://api.aigolife.com/if/account.jhtml?action=UserLogin&username=15850754914&password=e10adc3949ba59abbe56e057f20f883e&sign=ca84ad7a8c5e88cec077c31a72752c40

2.注册(需要验证码)：UserRegister2

  • 传入：
名称 类型 说明 可为空

sign String md5(接口完整url包含传入的参数(不包含参数sign)+’2013-03-29’+action).toLowerCase() 否
  • 返回：
名称 类型 说明 可为空
result ResultObject 执行结果 否
user UserObject 注册成功的用户对象 否
  • 接口地址：http://api.aigolife.com/if/account.jhtml
